# **App Name**: Aura Launcher

## Core Features:

- Icon Pack Support: Allows the user to apply icon packs from the Google Play Store.
- Customizable Grid: Provides settings for grid layout, icon size, and spacing.
- Widget Management: Simplifies widget placement, sizing, and organization on the home screen.
- Theme Engine: Provides users the ability to customize theme and automatically switch to dark theme at specific times
- Save User Config: Allows the user to save configurations and settings.

## Style Guidelines:

- Primary color: Dark slate blue (#434A54) for a professional and sophisticated look in dark mode.
- Background color: Very dark gray (#2A2E35) for a comfortable, modern dark mode.
- Accent color: Electric purple (#DF00FF) to highlight interactive elements against the dark background.
- Body and headline font: 'Inter', a grotesque-style sans-serif font for a clean, modern aesthetic.
- Use flat, minimalist icons that are easily recognizable, with consistent styling.
- Maintain a clean and spaced layout with ample negative space, ensuring each UI element is distinct and easily tappable.
- Implement subtle, smooth transitions for app opening, closing, and screen changes, creating a fluid user experience.
